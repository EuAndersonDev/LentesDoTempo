# Lentes do Tempo - Documentacao Frontend

Este diretorio centraliza a documentacao tecnica do frontend.

## Indice

1. auth-routes.md
   - Fluxo de autenticacao e recuperacao de senha.
   - Mapeamento de paginas, scripts e endpoints.

2. README-AUTH.md
   - Copia da documentacao original de autenticacao.
   - Referencia completa de fluxo e payloads.

3. api-integration.md
   - Integracao frontend com backend.
   - Modulo central em /api/config.js.

4. frontend-architecture.md
   - Estrutura de pastas, convencoes e responsabilidades.

5. run-and-config.md
   - Como executar o frontend localmente.
   - Configuracoes necessarias para comunicar com o backend.

6. speech-api-readme.md
   - Documentacao completa da Speech API.
   - Reconhecimento de voz em portugues (PT-BR).
   - Comandos de navegacao e controle do site.

7. speech-api-quickstart.md
   - Guia rapido para usar a Speech API.
   - Instalacao em 2 minutos.
   - Exemplos de comandos e personalizacao.

8. speech-api-implementation.md
   - Status de implementacao da Speech API.
   - Lista completa de arquivos criados e paginas integradas.
   - Estatisticas e checklist de conclusao.

## Escopo

- Esta documentacao e focada no app web estatico em LentesDoTempo.
- O backend correspondente esta documentado em Backend/docs.
- A conexao HTTP do frontend com o backend deve ser centralizada em /api/config.js.
- A Speech API (reconhecimento de voz) foi implementada em 18/04/2026 com suporte em todas as paginas.
